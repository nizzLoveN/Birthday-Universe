# A Little Universe For You 🌌💖

**A Premium Digital Birthday Experience**
Sebuah template website romantis interaktif kelas premium yang dirancang khusus sebagai hadiah ulang tahun. Dibangun menggunakan arsitektur Vanilla HTML5, CSS3, dan JS ES6 tanpa membebani performa browser dengan framework atau library berat.

## 🎨 Konsep Desain
*   **Vibe:** Premium Aesthetic, Soft Luxury, Pinterest Inspired.
*   **Warna Dominan:** Soft Peach Pink, Blush Pink, Clean White, Elegant Dark Contrast.
*   **Fitur Utama:** Background Music Player, Interactive Story Timeline, Polaroid Memory Gallery, 30 Quotes Generator, Catch The Hearts Game, Digital Gift Box, dan Surat Cinta Interaktif.
